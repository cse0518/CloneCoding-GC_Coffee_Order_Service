# 커피 주문 관리 어플리케이션
<br/>

## 프로젝트 목적
- 관리자가 커피 상품을 등록 및 조회 할 수 있는 서비스 구현
- 고객이 커피 상품을 주문할 수 있는 서비스 구현
- React로 구현된 Frontend 소스를 받아 Backend 구현
<br/>

## 프로젝트 환경
- Language : `JAVA 16`
- Framework : `SpringBoot 2.5.4`
- Build Tool : `Maven`
- Database : `MySQL 8`
- OS : `Windows 10`
<br/>
